# Gerenciador de Dívidas

## Descrição

O **Gerenciador de Dívidas** é uma aplicação web simples desenvolvida com **HTML, CSS e JavaScript**, que permite registrar pessoas e valores de dívidas, calcular o total e organizar as informações de forma prática.

O objetivo do projeto é praticar conceitos de manipulação do DOM, validação de dados e criação dinâmica de elementos utilizando JavaScript puro.

## Funcionalidades

* Adicionar pessoas com valores de dívidas
* Validação de campos (nome e valor)
* Cálculo automático do valor total das dívidas
* Exibição dinâmica das informações na tela
* Interface simples e fácil de usar

## Tecnologias Utilizadas

* HTML
* CSS
* JavaScript

## Como Executar o Projeto

1. Clone o repositório:

```bash
git clone https://github.com/RafaelaDesousa33/projeto_gerenciador_dividas.git
```

2. Acesse a pasta do projeto:

```bash
cd projeto_gerenciador_dividas
```

3. Abra o arquivo:

```bash
index.html
```
